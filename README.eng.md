## 💎 About

The project was developed for fixing concepts in React.

<h3>Operation</h3>

The <b>My Contact</b> App is a list of contacts made in <b><i>JavaScript</b></i> that is consumed by modules that distribute them in an orderly manner in the Template.
To do this we used the componentization of the information and the <b><i>map()</b></i> method to map and distribute it, creating a <b>Card</b> for each contact. 
The contact information is:

- [x] Nome
- [x] Imagem
- [x] Telefone 
- [x] E-mail

They are distributed in the following components:

- Card (containing all information, consuming the Avatar and Details components)
- Avatar (containing the image)
- Details (containing the details such as phone number and e-mail address)

## ⚙️ Execution

After downloading the project, in the terminal, type the following command to run the project:

```bash
npm start
```

Once the above command is executed, the project will be available for execution at the following address:

```bash
http://localhost:3000
```

Open the address in your favorite browser.
